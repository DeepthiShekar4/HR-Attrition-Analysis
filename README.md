# HR Attrition Analysis Dashboard (Tableau)

# Project Overview
Employee attrition has a direct impact on organizational performance and workforce stability.
This project presents an interactive HR Attrition Analysis Dashboard built using Tableau to analyze employee attrition patterns and identify key factors influencing employee turnover.
The dashboard converts raw HR data into meaningful insights by examining attrition trends across departments, job roles, demographics, satisfaction levels, and tenure.
